# Archive

Early-iteration kernel implementations and profiling artifacts from the initial exploration phase.

These files use an older naming convention (Config A–D, mixed naming patterns) and are preserved here for reference. The canonical implementations and results now live in the main operation directories using the standardized `configG`/`configH`/`mixedMP` naming.

## Contents

- `kernels/` — Standalone matmul scripts from initial development (BMM variants, test harnesses)
- `results/` — NCU profiling reports from early config sweeps (Config A–D, mixed precision exploratory runs)
